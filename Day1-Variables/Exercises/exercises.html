Exercises: Variables

Here are some practice problems for you to try after reading the Medium post:

  1. Create a variable called favoriteFruit and assign it your favorite fruit. Print it in the console.
  2. Change the value of favoriteFruit to another fruit and print it again.
  3. Create a variable called age and assign it your current age. Print it.
  4. Imagine you are in a science lab. Create a variable called testTube and assign it any liquid sample (string). Print it.
  5. Create a variable called shoppingCart and update it three times with different grocery items, printing the cart each time.
  6. Use const to create a variable birthYear with your year of birth. Try to reassign it and observe what happens.
  7. Declare a variable without assigning any value to it. Print it to see what shows up by default.
  8. Use typeof to check the data type of your age and favoriteFruit variables. Print the results.
  9. Create two variables, firstName and lastName, then combine them into a fullName variable. Print it.
  10. Bonus: Swap the values of two variables (a and b) using a third temporary variable. Print the result.
