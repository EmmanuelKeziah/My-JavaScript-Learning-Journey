Day 2: JavaScript Data Types

Practice the following exercises before checking the solutions.

  1. Declare a string variable and log its length.
  2. Convert a number to a string.
  3. Check the type of a boolean value.
  4. Show what happens when a variable is declared but not initialized.
  5. Assign null to a variable and check its type.
  6. Create two symbols with the same description and compare them.
  7. Store a very large integer using BigInt.
  8. Create an object with three properties.
  9. Make an array of fruits and access the second element.
  10. Write a function that prints a greeting.
  11. Show dynamic typing: assign different types to one variable.
  12. Concatenate a string and number.
  13. Convert a boolean to a number.
  14. Check if a variable is an array.
  15. Add a new element to an array.
  16. Access object properties using dot and bracket notation.
  17. Write a function that returns the type of any input.
  18. Compare null and undefined.
  19. Create a nested object and access a value inside.
  20. Create an array of objects and access one property.
